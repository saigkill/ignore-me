ignore-me 0.1.2

Release Notes

ignore-me is a generator for ignoring files for Autotools based projects and it knows how to handle autoconf, automake, libtool, gtk-doc, gnome-doc-utils, yelp.m4, mallard, intltool, gsettings, dejagnu, appdata, appstream.

New Features #

Enhancements #

  • Fixed Bug LP#1732232: Implemented a updating option for the Makefiles. The following commands now can be used:

    • make git-mk-update

    • make bzr-mk-update

    • make svn-mk-update

    • make cvs-mk-update

    • make hg-mk-update

Notable Bugs fixed: #

  • Bug LP#1732229: Add missing copyright notes to debian/copyright. (Needed for releasing in Debian Distro)

  • Bug LP#1732230: Upgrade packaging base:

    • debian/control:

      • Set Standards version to 4.1.1

      • Debhelper updated to depend on version 10

    • debian/compat

      • Update to level 10

  • Bug LP#1732232: Use new git location for update src files (See Enhancements).
