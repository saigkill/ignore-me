How to use ignore-me

Generator for ignoring files for Autotools based projects

ignore-me is a generator for ignoring files for autotools based projects and it knows how to handle autoconf, automake, libtool, gtk-doc, gnome-doc-utils, yelp.m4, mallard, intltool, gsettings, dejagnu, appdata, appstream.
