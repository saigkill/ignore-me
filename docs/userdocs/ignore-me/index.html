How to use ignore-me

Generator for ignoring files for Autotools based projects

ignore-me is a generator for ignoring files for autotools based projects and it knows how to handle autoconf, automake, libtool, gtk-doc, gnome-doc-utils, yelp.m4, mallard, intltool, gsettings, dejagnu, appdata, appstream.

Installation #

This section describes, how to install ignore-me for Ubuntu, Debian or openSUSE.

Installation on Debian #

The installation works in the console/terminal via: sudo apt install ignore-me (in unstable).

Installation on Ubuntu #

Do the following to get ignore-me get work:

  1. Type in the console: sudo add-apt-repository ppa:ignore-me/ppa and press Enter.

  2. Now type: sudo apt update && sudo apt install ignore-me and press Enter.

Installation on openSUSE #

To install ignore-me on openSUSE, follow the next steps:

  1. Type sudo zypper ar https://download.opensuse.org/repositories/home:/saigkill:/ignore-me/openSUSE_Leap_42.2 for Leap 42.2 and press Enter.

    Type sudo zypper ar https://download.opensuse.org/repositories/home:/saigkill:/ignore-me/openSUSE_Leap_42.3 for Leap 42.3 and press Enter.

    Type sudo zypper ar https://download.opensuse.org/repositories/home:/saigkill:/ignore-me/openSUSE_Tumbleweed for Tumbleweed and press Enter.

  2. Then type sudo zypper ref && sudo zypper in ignore-me and press Enter.

Usage #

ignore-me defines some common tasks in your Makefile. Now following the VCS specific steps to use ignore-me.

With BZR #

  1. Change inside the shell to your project's root directory.

  2. Type copy-bzrmk and press Enter.

  3. Type make -f bzr.mk and press Enter.

    Note

    The copy command modifies all Makefile.am files and includes itself there. This enables automatic .bzrignore generation.

  4. Define a BZRIGNOREFILES variable into your Makefile.am, and add the files who should be ignored.

  5. Add to MAINTAINERCLEANFILES in your Makefile.am:

    • $(BZRIGNORE_MAINTAINERCLEANFILES_TOPLEVEL)

    • $(BZRIGNORE_MAINTAINERCLEANFILES_MAKEFILE_IN)

    • $(BZRIGNORE_MAINTAINERCLEANFILES_M4_LIBTOOL)

Common Information #

If you need to ignore more files, add them to the BZRIGNOREFILES, GITIGNOREFILES, HGIGNOREFILES, SVNFILES or CVSFILESvariable in your Makefile.am.

But think twice before doing that. If a file has to be in a .xxxignore, chances are very high that it's a generated file and should be in one of MOSTLYCLEANFILES, CLEANFILES, DISTCLEANFILES, or MAINTAINERCLEANFILES.

The only case that you need to manually add a file to XXXIGNOREFILES is when remove files in one of mostlyclean-local, clean-local, distclean-local, or maintainer-clean-local make targets.
