7 Usage

7.5 With CVS

ignore-me defines some common tasks for your Makefile.am. Now following the VCS specific steps to use ignore-me.

With BZR #

  1. Change inside the shell to your project's root directory.

  2. Type tux > copy-bzrmk and press Enter.

  3. Type tux > make -f bzr.mk and press Enter.

    Note

    The copy command modifies all Makefile.am files and includes itself there. This enables automatic .bzrignore generation.

  4. Define a BZRIGNOREFILES variable into your Makefile.am and add the files who should be ignored.

  5. Add to MAINTAINERCLEANFILES in your Makefile.am:

    • $(BZRIGNORE_MAINTAINERCLEANFILES_TOPLEVEL)

    • $(BZRIGNORE_MAINTAINERCLEANFILES_MAKEFILE_IN)

    • $(BZRIGNORE_MAINTAINERCLEANFILES_M4_LIBTOOL)
