6 Installation

6.1 Installation on Debian
6.2 Installation on Ubuntu
6.3 Installation on openSUSE
6.4 Generic Installation

This section describes, how to install ignore-me for Ubuntu, Debian or openSUSE.

Installation on Debian #

The installation works in the console/terminal via (with root): root # sudo apt install ignore-me and press Enter (in unstable).

Installation on Ubuntu #

Do the following to get ignore-me get work:

  1. Type in the console (as root): root # sudo add-apt-repository ppa:ignore-me/ppa and press Enter.

  2. Now type: root # sudo apt update && sudo apt install ignore-me and press Enter.

Installation on openSUSE #

To install ignore-me on openSUSE, follow the next steps (as root):

  1. Type root # sudo zypper ar https://download.opensuse.org/repositories/home:/saigkill:/ignore-me/openSUSE_Leap_42.2 for Leap 42.2 and press Enter.

    Type root # sudo zypper ar https://download.opensuse.org/repositories/home:/saigkill:/ignore-me/openSUSE_Leap_42.3 for Leap 42.3 and press Enter.

    Type root # sudo zypper ar https://download.opensuse.org/repositories/home:/saigkill:/ignore-me/openSUSE_Tumbleweed for Tumbleweed and press Enter.

  2. Then type root # sudo zypper ref && sudo zypper in ignore-me and press Enter.

6.4 Generic Installation #

For all other distros are the following steps proposed:

  1. tux > tar -xf ignore-me-0.1.2.tar.xz

  2. tux > cd ignore-me-0.1.2

  3. tux > ./configure

  4. tux > make

  5. [ Become root if necessary ]

  6. root # make install
