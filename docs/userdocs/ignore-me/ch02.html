2 Feedback

Several feedback channels are available:

Bug Reports & Feature Requests

You are welcome to file a Bug Report or a Feature Request.

File a bug: https://bugs.launchpad.net/ignore-me (https://bugs.launchpad.net/ignore-me).

File a feature request: https://blueprints.launchpad.net/ignore-me (https://blueprints.launchpad.net/ignore-me).

User Comments & Ask a question

We want to hear your comments about and suggestions for this manual and the other documentation included with this product. Go to https://answers.launchpad.net/ignore-me (https://answers.launchpad.net/ignore-me) and enter your comments there.

Mail

For feedback on the documentation of this product, you can also send a mail to Sascha.Manns@mailbox.org (mailto:Sascha.Manns@mailbox.org). Make sure to include the document title, the product version and the publication date of the documentation. To report errors or suggest enhancements, provide a concise description of the problem and refer to the respective section number and page (or URL).

Release Notes

You can view the Release Notes on: https://saigkill.github.io/ignore-me/relnotes/ignore-me-release-notes/index.html
